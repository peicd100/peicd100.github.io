

### **數學符號**
| 符號         | LaTeX 指令 | 說明     |
| ------------ | ---------- | -------- |
| \(+\)        | `+`        | 加號     |
| \(-\)        | `-`        | 減號     |
| \(\times\)   | `\times`   | 乘號     |
| \(\cdot\)    | `\cdot`    | 內積     |
| \(\div\)     | `\div`     | 除號     |
| \(=\)        | `=`        | 等號     |
| \(\neq\)     | `\neq`     | 不等號   |
| \(\leq\)     | `\leq`     | 小於等於 |
| \(\geq\)     | `\geq`     | 大於等於 |
| \(\infty\)   | `\infty`   | 無窮大   |
| \(\sqrt{x}\) | `\sqrt{x}` | 開平方根 |
| \(\sum\)     | `\sum`     | 求和     |
| \(\int\)     | `\int`     | 積分     |
| \(\partial\) | `\partial` | 偏導符   |
