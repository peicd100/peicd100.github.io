
# 懶人輸入法，比賽時用
> 以下為[APCS在2023/10/22的第三題](https://zerojudge.tw/ShowProblem?problemid=m372)測資

```cpp
輸入：
3 4
FHH7
IIII
LHHJ

輸出：
10

輸入：
4 7
0F70000
FXJ0000
II700X7
LJ0HHLJ

輸出：
9
```
> APCS提供的 Codeblocks 不支援複製、貼上 ，對於打字慢的人非常不友善。
像這個題目，測資一大串，還英文與數字混和，如果我們想要一直測試的話可以怎麼辦呢?



>看到程式碼，我們可以將cin替換成自己的cin，然後在我們的cin放入要輸入的資料


```cpp  
//istream& ss = cin;
#include <bits/stdc++.h>
using namespace std;
int main(){

    istringstream cin("4 7 \
0F70000 \
FXJ0000 \
II700X7 \
LJ0HHLJ");//註解這一行
    
    int n,m;
    string s;
    cin>>n>>m;
    for(int i=0;i<n;i++){
        cin>>s;
        cout<<s<<"\n";
    }

}
```

///

>將輸入放入cin("")中，如果想要資料換行，要在結尾使用`\`反斜線。


>但是要注意
```
"123 \
456"
```
和
```
"123 456"
```
是一樣的，不要因為撰寫時的格式換行就以為他有換符號   
也就是說不會變成以下：
```
"123 \n 456"
```
