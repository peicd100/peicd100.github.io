## 嵌入PDF

### GOOGLE雲端

<details>
<summary>預覽</summary>

<iframe src="https://drive.google.com/file/d/1KLN8uow9WGAGrZwYADxiWyUsJ8w6_2VV/preview" width="100%" height="1100px"></iframe>

</details>

#### 代碼：
```html
<iframe src="https://drive.google.com/file/d/1KLN8uow9WGAGrZwYADxiWyUsJ8w6_2VV/preview" width="100%" height="1000px"></iframe>
```

> 若網址為
> `https://drive.google.com/file/d/1KLN8uow9WGAGrZwYADxiWyUsJ8w6_2VV/view?usp=drive_link`  
> 將最後一個斜線後面「`view?usp=drive_link`」改為「`preview`」，如下：  
> `https://drive.google.com/file/d/1KLN8uow9WGAGrZwYADxiWyUsJ8w6_2VV/preview`

#### 格式
```html
<iframe src="網址" width="100%" height="1000px"></iframe>
```
