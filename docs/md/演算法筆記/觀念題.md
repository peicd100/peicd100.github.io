### 1. 遞迴函式輸出

**選項**

* A. 229 8
* B. 230 7
* C. 230 8
* D. 228 7

**題目：請問以下程式碼輸出為何**

```cpp
#include <bits/stdc++.h>
using namespace std;

int g(int x,int y);
int f(int x,int y);
int t;

int g(int x,int y){
    if(x==0 && y==1)t++;
    if(x<=0 || y<=0)return 3;
    return 5+f(x,y-1);
}
int f(int x,int y){
    if(x<=0 || y<=0)return 2;
    return g(x-1,y)+2*g(x-2,y-1);
}

int main(){
    printf("%d",f(5,5));
    printf("\n");
    printf("%d",t);
    return 0;
}
```

**答案：A**

### 5. Bubble Sort 交換次數

**選項**

* A. 14
* B. 15
* C. 16
* D. 17

**題目：請問以下程式碼輸出為何**

```c
#include <stdio.h>

int t=0;

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; ++i) {
        for (int j = 0; j < n - i - 1; ++j) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                t++;
            }
        }
    }
}

int main() {
    int arr[]={10,3,2,1,4,6,5,9,7,8}, n=10;
    bubbleSort(arr, n);
    printf("%d",t);
    return 0;
}
```

**答案：B**
