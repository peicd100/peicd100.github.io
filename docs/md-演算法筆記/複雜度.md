
# 複雜度

## 總結

自己寫一個小程式
令 $O(f(n))$ 是時間複雜度
算算看 `n ="範圍"` 帶入 $f(n)$ 是否會超過 7 位數 (7 位數是可接受的)
也就是 $f(n)<1*10^8=1e8$

## 自己

口訣：一二五百千萬

| Big-O 複雜度 | 競賽適用範圍                     |
|--------------|----------------------------------|
| $O(n!)$      | $n \leq 10$                     |
| $O(2^n)$     | $n \leq 20$                     |
| $O(n^4)$     | $n \leq 50$                     |
| $O(n^3)$     | $n \leq 百位數$                    |
| $O(n^2)$     | $n \leq 千位數$                   |
| $O(n \log n)$| $n \leq 萬以上$ |



## IONC

每秒 $10^7$

| Big-O 複雜度 | 競賽適用範圍                     |
|--------------|----------------------------------|
| $O(n!)$      | $n \leq 10$                     |
| $O(2^n)$     | $n \leq 20$                     |
| $O(n^4)$     | $n \leq 50$                     |
| $O(n^3)$     | $n \leq 200$                    |
| $O(n^2)$     | $n \leq 3000$                   |
